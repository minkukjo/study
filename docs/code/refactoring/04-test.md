## 테스트 구축하기

리팩토링을 하기 위해선 반드시 테스트가 존재해야한다.

## 자가 테스트 코드의 가치

- 테스트코드는 지루한 작업이지만 결과적으로는 프로그래밍의 개발 속도를 높여준다.

- 켄트백은 테스트코드부터 작성하는 일명 TDD를 창시했다.

- 리팩토링을 하고 싶다면 반드시 테스트 코드가 존재해야한다.
